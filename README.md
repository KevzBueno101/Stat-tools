# Stat-tools

A comprehensive collection of Python-based statistical analysis tools designed for freelance statistical assistance work.

## Overview

This repository contains statistical computation and validation tools that I use in my freelance work as a **Statistical Assistant**. These tools help streamline the process of checking, computing, and validating statistical data for various research and theses.

## Purpose

These tools are specifically used for:

- **Checking** - Verifying statistical calculations and data integrity
- **Computing** - Performing statistical analyses and calculations
- **Validating** - Ensuring accuracy and reliability of statistical results

## Usage

Each script is designed to be used independently based on the specific statistical analysis needed. Simply run the appropriate Python script for your analysis requirements.

## Technologies

- Python
- Statistical libraries (scipy, numpy, pandas)
- Jupyter Notebook for interactive analysis
